折角なので、英語版サイトも見てみましょう。

めっちゃシンプルです。

見ていくのは以下の3つです。

- 外観
- パフォーマンス
- 使用技術

ちなみに、私のスタンスとしては「見やすくて速ければ、どんな技術を使っていてもいい」と思っています。
新しい流行の技術を使えばよいページになる訳ではありません。
古い技術であっても、それで安定していればいいと思っています。

もちろん、Deprecated な API を使っていたり、
脆弱性のあるバージョンを使っていたらその限りではありません。
経験上、大きい所は作ったら作りっぱなしのところが多いと
なんとなく思っているので、今回はその辺も見ていきたいと思います。
